Main structure :
selector1, selector2 {
property : value ;
...
}

There are 4 types of Selectors
1.Elements
2.specific id : #
3.Class : a period .
4.Universal : * all html elements on page

# We can group selectors 

Examples
1. Elements
p{
color: red;
text-alilgn: center;
}

2.Id
<p id="para1">Hello World!</p>

#para1 {
  text-align: center;
  color: red;
}

3. Class
.center {
  text-align: center;
  color: red;
}

4. Universal
* {
  text-align: center;
  color: blue;
}

<h1 class="center">Red and center-aligned heading</h1>


# Group example

h1, h2, p{
...
}
